Versi 7.1.1

Overview

Muslimku is an NVDA add-on that provides prayer time announcements, Hijri/Gregorian date information, Qibla checking, and reminder notifications based on your selected location.

Ringkasan

Muslimku adalah add-on NVDA untuk pengumuman waktu salat, tanggal Hijriah/Masehi, pengecekan kiblat, dan notifikasi pengingat berdasarkan pengaturan lokasi.

Features

  • Daily prayer announcements: Fajr, Dhuhr, Asr, Maghrib, and Isha.
  • Additional time announcements: Imsak, Sunrise, Dhuha, and Sunset.
  • Next-prayer announcement with remaining time.
  • Current location announcement.
  • Qibla direction check in degrees and compass direction.
  • Hijri/Gregorian date announcement with Maghrib-based Hijri transition.
  • Automatic reminders with configurable minute offset.
  • Speech-first reminders with optional Windows toast notifications.
  • Double-press gesture support to copy announcement text to clipboard.
  • All gestures can be remapped from NVDA Input Gestures.
  • Calculation method selection is available in settings, including Muhammadiyah.
  • Asr madhab selection is available (Shafi/Hanafi).

Fitur

  • Pengumuman salat wajib: Subuh, Dzuhur, Ashar, Maghrib, Isya.
  • Pengumuman waktu tambahan: Imsak, Terbit, Dhuha, dan Terbenam.
  • Pengumuman salat berikutnya beserta sisa waktu.
  • Pengumuman lokasi saat ini.
  • Pengecekan arah kiblat dalam derajat dan arah mata angin.
  • Pengumuman tanggal Hijriah/Masehi dengan perpindahan Hijriah berbasis Maghrib.
  • Pengingat otomatis dengan offset menit yang bisa diatur.
  • Pengingat berbasis suara NVDA dengan notifikasi toast Windows opsional.
  • Tekan gesture dua kali untuk menyalin teks hasil pengumuman.
  • Semua gesture dapat diubah dari Input Gestures NVDA.

Location Settings

  • If the selected country is Indonesia, the add-on uses Province and City/Regency combo boxes.
  • If the selected country is not Indonesia, the add-on uses a City combo box list based on that country.
  • Language can be set to Indonesian or English.
  • Calculation method can be selected from settings.
  • Asr madhab can be selected: Shafi or Hanafi.
  • Reminder settings include enable/disable and reminder offset (time range in minutes).

Pengaturan Lokasi

  • Jika negara yang dipilih adalah Indonesia maka memakai combo box Provinsi dan Kota/Kabupaten.
  • Jika Negara yang dipilih Non Indonesia maka memakai daftar combo box Kota sesuai negara terpilih.
  • Bahasa dapat dipilih Indonesia atau English.
  • Pengaturan pengingat dapat aktif/nonaktif dan terdapat offset menit (rentang waktu) peringatan.
